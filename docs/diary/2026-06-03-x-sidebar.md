# x-sidebar Architecture

A simple dired sidebar for quick directory browsing and file preview.

## Overview

- **Entry**: `C-s C-w` (`M-x x-sidebar`)
- **Width**: 30 columns (customizable via `x-sidebar--width`)
- **Layout**: enforce 2-window — left sidebar, right main

## Data Structures

### State Alist: `x-sidebar--state`

```elisp
((frame1 . buffer1) (frame2 . buffer2) ...)
```

- Key: frame object (`selected-frame`)
- Value: dired buffer for that frame's sidebar
- Multi-frame safe: each frame has its own entry, created lazily via `x-sidebar--state-entry`

No window is stored — it is always found by scanning buffer names.

### Window Discovery: `x-sidebar--find-window`

Scans all windows on the current frame for one whose buffer name starts with `*x-sidebar*`. Returns the window or nil.

## Key Entry Point: `x-sidebar`

```
select-window win        ← sidebar already open on this frame
  or
dired-noselect           → create/setup dired buffer
delete-other-windows     → clear frame to single window
split-window ... 'left   → sidebar on left, original window on right
select-window sidebar    → focus sidebar
preview                  → update main window
```

## Navigation

| Key | Action | Implementation |
|-----|--------|----------------|
| `RET` / `l` / `<right>` | Enter dir or open file | `x-sidebar--ret` |
| `h` / `<left>` | Go to parent directory | `x-sidebar--up-directory` → `x-sidebar--enter-dir` |
| `j` / `<down>` | Next line | `dired-next-line` |
| `k` / `<up>` | Previous line | `dired-previous-line` |
| `C-s C-w` | Switch to main window | `x-sidebar--switch-to-main` |

### `x-sidebar--ret`

- **Directory on cursor**: calls `x-sidebar--enter-dir`
- **File on cursor**: `select-window` to main window, `find-file`, then back to sidebar

### `x-sidebar--enter-dir`

1. Expand directory path to absolute
2. Find existing dired buffer for that directory (`dired-buffers-for-dir`) or create new (`dired-noselect`)
3. Call `x-sidebar--setup` on buffer (keys, hide-details, post-command-hook)
4. Store as current sidebar buffer via `x-sidebar--set-buffer`
5. `set-window-buffer` to show in sidebar window
6. Trigger preview

### `x-sidebar--up-directory`

Computes parent directory from `default-directory`, enters it via `x-sidebar--enter-dir`, then jumps to the original child directory entry via `dired-goto-file`.

## Preview: `x-sidebar--preview`

Triggered by `post-command-hook` in the sidebar buffer.

```
dired-get-file-for-visit  → file path
skip if first-time or same as last  → `x-sidebar--last-preview`
directory → dired-noselect
file      → find-file-noselect
set-window-buffer on main window
```

The main window is found/created by `x-sidebar--main-window`.

## Main Window: `x-sidebar--main-window`

```elisp
existing non-sidebar window on current frame
  or
create new window:
  priority 1: file/dir at cursor in sidebar
  priority 2: other-buffer
split right with sidebar width preserved
```

| Key | Command |
